# Rubber Ducky Evidence

Store raw evidence for authorized Rubber Ducky/HID testing here.

## Evidence Types

- Screenshots of harmless baseline behavior.
- Endpoint device insertion logs.
- SIEM alerts or query results.
- Device inventory exports.
- Operator activity notes.

## Handling Rules

- Do not store credentials, tokens, cookies, API keys, or personal identifiers.
- Redact private hostnames, usernames, emails, phone numbers, IP addresses, and customer details before sharing.
- Keep raw evidence private unless publication is explicitly approved.
