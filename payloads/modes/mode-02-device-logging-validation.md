# Mode 02: Device Logging Validation

## Purpose

Help the blue-team project validate endpoint and SIEM visibility for new USB HID devices.

## Expected Behavior

- Pi Pico connects as a lab HID device.
- No keystrokes are sent.
- Operator records insertion and removal times.

## Evidence

- Endpoint USB/HID log entries.
- SIEM query or alert screenshot if available.
- Device identifiers when visible.
