# Mode 03: User Awareness Demo

## Purpose

Demonstrate to a lab or training audience why unknown USB devices should be reported.

## Expected Behavior

- Operator manually opens a text editor or training note.
- Pi Pico types only the approved awareness message.
- No commands or system changes.

## Approved Message

```text
Training demo: unknown USB devices can act like keyboards. Report unknown devices.
```
